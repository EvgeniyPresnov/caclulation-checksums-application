# Project Title 
The application for calculating the checking sum of files.

# Prerequisites
* java 8
* jdk 8

# Installing
```
 git clone git@github.com:ZhekaPresnov/JavaFXProject.git
 cd JavaFXProject
 mvn clean install
 mvn exec:java -Dexec.mainClass="ru.cniiag.app.gui.Main"
```

# Built With
* Maven - Dependency Management
